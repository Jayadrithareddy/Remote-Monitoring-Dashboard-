====================================
SIMPLE NODE-RED SETUP FOR LIVE DATA
====================================

🎯 GOAL: Get MQTT data into React Dashboard

📌 3 SIMPLE STEPS:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1: Add MQTT Input in Node-RED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open Node-RED: http://localhost:1880
2. Drag "mqtt in" node from left panel
3. Double-click it and configure:
   
   Server: 127.0.0.1:1883
   Topic: Jyo/data
   QoS: 0
   Output: auto-detect (JSON object)

4. Click "Done"


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2: Create HTTP Endpoint
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Drag "http in" node
2. Configure:
   Method: GET
   URL: /livedata

3. Drag "http response" node
4. Connect: [http in] → [http response]


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3: Store & Serve Data
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Drag "function" node between MQTT and HTTP
2. Name it: "Store Data"
3. Paste this code:

   flow.set('latestData', msg.payload);
   return msg;

4. Connect like this:

   [MQTT in] → [Store Function]
   
   [HTTP in] → [Get Function] → [HTTP response]

5. Add another "function" node (Get Data):

   msg.payload = flow.get('latestData') || {
     BatteryVoltage: 0,
     OilPressure: 0,
     EngineSpeed: 0
   };
   return msg;

6. Click DEPLOY (red button top-right)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
